# Connect Four (Pettingzoo + Rule-based)
A Connect Four game using the Pettingzoo environment, featuring:

- A human player vs AI
- Rule-based strategies
- Full board rendering in the terminal
- Move validation and win detection
- Clean, modular Python code

## AI Strategies & Baseline Comparison
This project implements multiple Connect Four AI strategies, ranging from a simple baseline to more advanced rule‑based agents. The goal is to compare their performance, understand their strengths and weaknesses, and allow players to challenge each strategy directly.

## Baseline Strategy
The baseline agent is intentionally simple:
- It selects a random valid column.
- It has no look‑ahead, no heuristics, and no tactical awareness.
- It serves as a reference point for evaluating more advanced strategies.
This baseline is useful because it shows how much improvement each rule‑based strategy provides.
